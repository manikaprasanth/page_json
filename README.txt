INTRODUCTION
------------

This module Provides a node as json format..

-- INSTALLATION --

* Install as you would normally install a contributed Drupal module. See:
   https://www.drupal.org/docs/8/extending-drupal-8/installing-drupal-8-modules
   for further information.

CONFIGURATION
-------------

Once the module has been installed, navigate to admin/config/system/site-information and configure the Site API Key.

Using Site API Key, We can get a node data as JSON format for page content type.

EX: sitename/apiKey/nid
